# Тестирование для социального контракта

Онлайн-система тестирования для подачи заявления на социальный контракт.

## Особенности

- 425 вопросов с рандомизацией
- 45 случайных вопросов за тест
- 3 варианта ответа на каждый вопрос
- Детальный разбор результатов
- Адаптивный дизайн

## Технологии

- HTML5, CSS3, JavaScript
- JSON для хранения вопросов
- Responsive design

## Запуск

Откройте `index.html` в браузере или разместите на хостинге.

## Структура

- `index.html` - главная страница
- `css/style.css` - стили
- `js/script.js` - основная логика
- `js/json-loader.js` - загрузчик вопросов
- `data/questions.json` - база вопросов
